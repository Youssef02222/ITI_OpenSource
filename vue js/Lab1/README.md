## Commands
- Install VueJS
```
$ npm i -g @vue/cli
```

- Create new project "lab1"
```
$ vue create lab1
```

- Run 
```
$ cd lab1
```
```
$ npm run serve
``` 

- Install bootstrap 
```
$ npm i bootstrap
```
